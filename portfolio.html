Ragnarok United Website

Sistema web para o servidor "Ragnarok United", consistindo de:

  • Layout (Bootstrap)
  • Sistema de cadastro com conta principal no site e contas de jogo, com modificações no emulador Hercules para uma melhor experiência
  • Integração com o sistema de fórum Invision Power Board (IPB) para exibição de últimas notícias e postagens do fórum
  • Funcionalidades de gerenciamento de conta
  • Sistema de cupons com prêmios no jogo
O servidor foi descontinuado em Junho/2018, portanto não há um link para visitá-lo.
